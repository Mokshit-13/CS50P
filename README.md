This repository contains my coursework, problem set solutions, and final project for **CS50’s Introduction to Programming with Python**, taught by David J. Malan at Harvard University. 

## 📖 About the Course

CS50P is a rigorous, comprehensive introduction to programming using Python. The curriculum focuses on teaching students how to read, write, test, and debug code. It emphasizes not just learning syntax, but understanding how to approach problems algorithmically and design elegant, efficient solutions.

## 🛠️ Skills & Technologies Demonstrated

Throughout this coursework, I have developed and applied skills in:
* **Core Python:** Functions, Variables, Conditionals, Loops, and Exceptions.
* **Libraries & APIs:** Utilizing third-party packages (via `pip`) and handling JSON data from web APIs.
* **Automated Testing:** Writing and executing unit tests using `pytest` to ensure code reliability.
* **File I/O:** Reading from and writing to text, CSV, and image files.
* **Data Extraction:** Utilizing Regular Expressions (`re`) to validate, extract, and format data.
* **Software Design:** Implementing Object-Oriented Programming (OOP) principles including classes, methods, inheritance, and operator overloading.

## 📂 Repository Structure

The repository is organized by the topics covered in the course's weekly problem sets:

* `0_functions_variables/`: Basic syntax, input/output, and string manipulation.
* `1_conditionals/`: Control flow and logical operators.
* `2_loops/`: `for` and `while` loops, iterating through lists and dictionaries.
* `3_exceptions/`: Error handling and input validation.
* `4_libraries/`: Modules, API requests, and command-line arguments.
* `5_unit_tests/`: Writing assertions and testing frameworks with `pytest`.
* `6_file_io/`: Data persistence using `.txt` and `.csv` files.
* `7_regular_expressions/`: Pattern matching and data validation.
* `8_object_oriented_programming/`: Classes, properties, and class methods.
* `project/`: My final capstone project.

## 🚀 Final Project: Command-Line To-Do List Manager

**Description:** A lightweight, interactive command-line application designed to help users organize their daily tasks. It allows for creating, viewing, updating, and deleting items on a to-do list directly from the terminal.

**Features:**
* **Persistent Storage:** Utilizes File I/O to save tasks to a file, ensuring data isn't lost between sessions.
* **Interactive CLI:** A clean, menu-driven interface using `while` loops and conditionals for seamless user navigation.
* **Robust Error Handling:** Implements `try-except` blocks to handle invalid user inputs and prevent the program from crashing.

**How to Run:**
```bash
# Clone the repository
git clone [https://github.com/yourusername/your-repo-name.git](https://github.com/yourusername/your-repo-name.git)

# Navigate to the project directory
cd your-repo-name/project

# Run the application
python project.py
```
